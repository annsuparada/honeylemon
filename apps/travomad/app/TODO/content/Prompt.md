# 📝 THE PROMPT TEMPLATE

```markdown
You are an expert SEO travel writer creating content for Honey Lemon.com, a travel blog that ranks highly on Google. Write a comprehensive, engaging, and SEO-optimized article following these exact specifications:

---

## ARTICLE DETAILS

**Topic:** [INSERT TOPIC - e.g., "Best Time to Visit Mexico"]

**Target Keyword:** [PRIMARY KEYWORD - e.g., "best time to visit Mexico"]

**Secondary Keywords:** [LIST 3-5 KEYWORDS - e.g., "Mexico weather by month", "when to go to Mexico", "Mexico travel seasons"]

**Target Audience:** [WHO IS THIS FOR - e.g., "First-time travelers to Mexico planning a beach vacation"]

**Word Count:** [1500-2500 words for cluster pages, 4000-5000 for pillar pages]

**Tone:** Helpful, conversational, enthusiastic but practical. Write like a knowledgeable friend giving advice, not a textbook. Use "you" to address the reader.

---

## CONTENT REQUIREMENTS

### 1. STRUCTURE

**Required Sections (in this order):**

**A. Introduction (200-250 words)**
- Hook the reader in the first sentence (surprising fact, question, or bold statement)
- Clearly state what the article will cover
- Explain why this matters to the reader
- Include target keyword in first 100 words
- End with a preview of what they'll learn

**B. Main Content (organized in clear H2/H3 sections)**
- Use descriptive H2 headers (not generic like "Overview")
- Each section should be 200-400 words
- Include specific, actionable information
- Use examples, comparisons, and real scenarios
- Break up long sections with H3 subheadings

**C. Practical Information (tables, lists, quick facts)**
- Add at least 2 visual elements (tables, comparison charts, or lists)
- Include specific prices, distances, or timeframes where relevant
- Make information scannable

**D. FAQ Section (3-5 questions)**
- Answer common questions readers might have
- Use question format for headers (H3)
- Keep answers concise (50-100 words each)

**E. Conclusion (100-150 words)**
- Summarize key takeaways (3-4 main points)
- Include a call to action
- Link back to pillar page
- End on an encouraging note

---

### 2. SEO OPTIMIZATION

**Keyword Usage:**
- Include target keyword in:
  - Title (H1)
  - First paragraph (within first 100 words)
  - At least 3 H2 headers
  - Meta description
  - Conclusion
  - 1-2 times per 300 words (naturally, not forced)

**Internal Linking:**
- Include [3-5] internal link placeholders formatted as: [LINK: Article Title]
- Link to pillar page in introduction and conclusion
- Link to 2-3 related cluster pages in main content
- Use descriptive anchor text (not "click here")

**Example Internal Link Placeholders:**
- "For more information, check out our [LINK: Mexico Travel Guide]"
- "Learn about [LINK: Best All-Inclusive Resorts in Mexico]"
- "Read our complete guide to [LINK: Cancun vs Tulum]"

---

### 3. WRITING STYLE REQUIREMENTS

**DO:**
✅ Write in active voice (not passive)
✅ Use short sentences (15-20 words average)
✅ Break paragraphs every 3-4 sentences
✅ Include specific details (exact prices, names of places, real examples)
✅ Use transition words (however, therefore, additionally, meanwhile)
✅ Address the reader as "you"
✅ Include personal recommendations ("Our top pick...", "We recommend...")
✅ Use power words (ultimate, essential, proven, exclusive, amazing)
✅ Add sensory details (colors, textures, sounds, tastes)
✅ Include actionable tips (numbered steps, specific advice)

**DON'T:**
❌ Use overly complex vocabulary
❌ Write long, academic paragraphs
❌ Make generic statements without specifics
❌ Use passive voice excessively
❌ Repeat the same keyword unnaturally
❌ Include fluff or filler content
❌ Use clichés ("hidden gem" used sparingly)
❌ Write in first person plural ("we think") unless giving personal experience

---

### 4. CONTENT DEPTH REQUIREMENTS

**For Each Major Section, Include:**

1. **Specific Facts & Data:**
   - Exact prices (not "cheap" or "expensive")
   - Specific locations/names (not "many places")
   - Timeframes (not "a while")
   - Statistics when relevant

2. **Comparisons:**
   - Compare 2-3 options
   - Show pros and cons
   - Help reader make decisions

3. **Practical Tips:**
   - What to bring
   - How to save money
   - Common mistakes to avoid
   - Best practices

4. **Real Examples:**
   - Sample itineraries
   - Actual hotel names
   - Specific restaurants or attractions
   - Real traveler scenarios

---

### 5. VISUAL ELEMENTS TO INCLUDE

Add placeholders for these visual elements:

**[TABLE: Title]** - For comparisons, data, or quick reference
**[LIST: Title]** - For steps, rankings, or tips
**[INFOGRAPHIC SUGGESTION: Description]** - For complex information
**[IMAGE SUGGESTION: Description + Alt Text]** - Every 300-400 words

Example:
```
[TABLE: Mexico Weather by Month]
| Month | Avg Temp | Rainfall | Crowds | Best For |
|-------|----------|----------|---------|----------|
```

---

### 6. FORMATTING REQUIREMENTS

**Headers:**
- H1 (Title): Include target keyword, make it compelling (50-60 characters)
- H2 (Main Sections): Descriptive, benefit-focused (not generic)
- H3 (Subsections): Specific, action-oriented

**Paragraphs:**
- Maximum 4 sentences per paragraph
- One idea per paragraph
- Mix short (1-2 sentence) and longer (3-4 sentence) paragraphs

**Lists:**
- Use numbered lists for steps/rankings
- Use bullet points for features/tips
- Minimum 3 items, maximum 10 per list

**Bold/Emphasis:**
- Bold key takeaways (1-2 per section)
- Bold important warnings or tips
- Don't overuse (max 2-3 per 100 words)

---

## ARTICLE-SPECIFIC INFORMATION

**Pillar Page This Belongs To:**
[PILLAR NAME - e.g., "Mexico Travel Guide"]
[PILLAR URL PLACEHOLDER - e.g., /destinations/mexico]

**Related Cluster Pages to Link To:**
1. [CLUSTER PAGE 1 - e.g., "Best All-Inclusive Resorts in Mexico"]
2. [CLUSTER PAGE 2 - e.g., "Mexico Travel Costs"]
3. [CLUSTER PAGE 3 - e.g., "Is Mexico Safe?"]

**Cross-Cluster Links (if applicable):**
- [CROSS-LINK 1 - e.g., "Budget Travel Guide" pillar]
- [CROSS-LINK 2 - e.g., "Best Beach Destinations" pillar]

---

## SPECIAL INSTRUCTIONS FOR THIS ARTICLE

[ADD ANY SPECIFIC REQUIREMENTS, such as:]
- Focus on families with young children
- Include budget-friendly options only
- Emphasize safety concerns
- Compare 3 specific destinations
- Include month-by-month breakdown
- Target luxury travelers
- etc.

---

## OUTPUT FORMAT

Please provide:

1. **Article Title (H1)**
2. **Meta Description (155 characters max)**
3. **Full Article Content** (following all requirements above)
4. **Internal Link Suggestions** (list of 5-8 links with anchor text)
5. **Image Suggestions** (list of 5-8 images with alt text)
6. **FAQ Schema** (JSON-LD format for the FAQ section)

---

## FINAL CHECKLIST

Before finishing, ensure the article has:
- [ ] Target keyword in title, intro, and conclusion
- [ ] 1500-2500 word count achieved
- [ ] 3-5 internal link placeholders included
- [ ] At least 2 tables, lists, or comparison charts
- [ ] FAQ section with 3-5 questions
- [ ] Specific, actionable information (not generic)
- [ ] Conversational, engaging tone
- [ ] Short paragraphs (3-4 sentences max)
- [ ] Proper header hierarchy (H1 > H2 > H3)
- [ ] No keyword stuffing (natural usage)

---

Now write the article following ALL requirements above.
```
