### 2. On-Page SEO Basics
- [ ] **Title Tags** - Each page needs unique, keyword-rich titles (50-60 chars)
  - Example: "12 Best All-Inclusive Resorts Mexico 2025 | Travomad"
- [ ] **Meta Descriptions** - Compelling summaries (150-160 chars)
  - Example: "Find the perfect all-inclusive resort in Mexico. Expert reviews of top properties in Cancun, Riviera Maya & Los Cabos. Compare prices & book now."
- [ ] **H1 Tags** - One per page, matches search intent
- [ ] **Header Hierarchy** - Proper H2, H3, H4 structure
- [ ] **URL Structure** - Clean, keyword-rich URLs (you already have this ✅)
- [ ] **Image Alt Text** - Describe every image for SEO + accessibility
  - Example: "Grand Velas Riviera Maya infinity pool overlooking Caribbean Sea"

### 3. Site Speed Optimization
- [ ] **Image Optimization**
  - Compress images (use WebP format)
  - Lazy loading for images
  - Proper image dimensions (don't load 4K images for 400px display)
- [ ] **Core Web Vitals**
  - LCP (Largest Contentful Paint): < 2.5s
  - FID (First Input Delay): < 100ms
  - CLS (Cumulative Layout Shift): < 0.1
- [ ] **Minify CSS/JS**
- [ ] **Enable caching**
- [ ] **Use CDN** (you're already using Cloudinary ✅)

### 4. Mobile Optimization
- [ ] Responsive design (test on all devices)
- [ ] Mobile-friendly navigation
- [ ] Touch-friendly buttons (min 48x48px)
- [ ] Fast mobile load times

### 5. Internal Linking Strategy
- [ ] Link blog posts to related content
- [ ] Create topic clusters (pillar pages + cluster content)
- [ ] Add "Related Posts" sections
- [ ] Footer links to important pages
- [ ] Breadcrumb navigation (links, not just schema)

### 6. XML Sitemap

### 2. Affiliate Link Strategy

#### Link Placement Best Practices
- [ ] **Within first 300 words** - "Book your stay at Grand Velas here"
- [ ] **In comparison tables** - Direct "View Prices" buttons
- [ ] **Call-to-action boxes** - Highlighted booking sections
- [ ] **Image captions** - "Click to see current rates"
- [ ] **Pros/Cons lists** - "Check Availability" links

#### Link Types to Use
- [ ] Text links (naturally in content)
- [ ] Button CTAs ("Check Rates", "Book Now", "View Deals")
- [ ] Image links (resort photos that are clickable)
- [ ] Price comparison tables with affiliate links

#### Disclosure Requirements (REQUIRED BY LAW)
- [ ] Add affiliate disclosure at top of every post
  - Example: "This post contains affiliate links. We may earn a commission when you book through our links at no extra cost to you."
- [ ] Add disclosure page in footer
- [ ] Mark affiliate links with rel="sponsored" or rel="nofollow"

### 3. Conversion Optimization

#### Create High-Converting Elements
- [ ] **Comparison Tables** - Side-by-side resort features
- [ ] **Price Alerts** - "Prices from $XXX/night"
- [ ] **Urgency Triggers** - "Limited availability", "Book now"
- [ ] **Trust Badges** - "Verified reviews", "Best price guarantee"
- [ ] **Social Proof** - "Over 5,000 travelers booked through us"

#### CTAs (Call-to-Actions)
Use strong action words:
- ❌ Weak: "Click here"

## Common Mistakes to Avoid

### SEO Mistakes:
- ❌ Keyword stuffing (sounds robotic)
- ❌ Duplicate content
- ❌ Slow website speed
- ❌ No mobile optimization
- ❌ Ignoring analytics

### Affiliate Mistakes:
- ❌ Too many affiliate links (looks spammy)
- ❌ Promoting products you haven't researched
- ❌ No disclosure (illegal)
- ❌ Only writing for commissions (readers can tell)
- ❌ Not tracking what works

### Content Mistakes:
- ❌ Inconsistent publishing
- ❌ Too short (< 800 words)
- ❌ No images
- ❌ Bad grammar/spelling
- ❌ Not solving reader problems
